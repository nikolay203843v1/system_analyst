1.2 Исправления:
- Если пользователь добавит больше 10 единиц одного товара — ошибка.

- Пусть товар удаляется, если количество 0. Кнопка удаления тоже пусть будет — для быстрого удаления, чтобы не крутить до нуля.

- Лимиты: максимум 5 разных товаров. Общее количество всех товаров — не больше 20 штук. Если один тип товара 3 штуки, то другие типы можно добавить так, чтобы общее не превышало 20.

- Показывать: цену за единицу товара, общую стоимость позиции (количество × цена), и в конце — общую стоимость всей корзины.

- Реклама пусть будет снизу при пролистывании — каталог товаров со скидками.

- Часовой пояс брать у пользователя.

- Цена должна обновляться в корзине, когда меняется цена товара.

- Реклама: где именно на странице? Точное время? Часовой пояс?
